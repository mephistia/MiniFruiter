Instruções de uso:
- Abrir a pasta "HelloGLFW-Box2D"
- Abrir a pasta "Debug"
- Executar o arquivo "HelloGLFW.exe"

Tecla C - Cria um círculo dinâmico aleatoriamente.
Tecla L - Cria uma linha estática aleatoriamente.
Tecla B - Cria uma caixa de tamanho aleatório no centro da tela, com coeficientes de atrito e restituição aumentando em 0.1 a cada caixa criada.
